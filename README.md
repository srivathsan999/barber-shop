# BarberX - Premium Barber Shop Website Template

A commercial-grade, premium HTML template for a Barber Shop & Grooming Studio.

## Features

- **Premium Design**: Dark/Light mode support with glassmorphism and modern aesthetics.
- **Responsive**: Built with Bootstrap 5 for perfect mobile/desktop rendering.
- **Animations**: Subtle GSAP animations for a polished feel.
- **Pages**: 13 full HTML pages including Home variants, Services, Gallery, and Auth pages.

## Usage

1. Open `index.html` in your browser.
2. Use the Moon/Sun icon in the navbar to toggle themes.
3. Edit the HTML files to update content.
